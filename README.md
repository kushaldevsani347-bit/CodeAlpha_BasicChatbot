# CodeAlpha Basic Chatbot

## Description

A simple rule-based chatbot developed in Python as part of the CodeAlpha Python Programming Internship.

## Features

- Greets users
- Responds to predefined questions
- Displays current date and time
- Tells jokes and motivational quotes
- Provides a help menu
- Handles unknown inputs

## Technologies Used

- Python 3

## How to Run

```bash
python chatbot.py
```
